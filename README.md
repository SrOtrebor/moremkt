# MoreMKT - Landing Page "Próximamente"

Esta es la página web de aterrizaje (Landing Page) en modo "Próximamente" o "En construcción" para **MoreMKT** (Estrategia de Paid Media y Performance). 
Fue diseñada enfocándose en una estética profesional, premium y dinámica, basándose en los lineamientos gráficos de la marca.

## 🚀 Tecnologías Utilizadas

- **HTML5:** Estructura semántica del contenido.
- **CSS3 Vainilla:** Estilos personalizados, gradientes y animaciones avanzadas (sin frameworks externos para asegurar máximo rendimiento y compatibilidad).
- **Google Fonts:**
  - `Outfit` para los títulos principales (look moderno).
  - `Inter` para los textos de lectura (alta legibilidad).
- **Phosphor Icons:** Para la iconografía minimalista y moderna de los botones de contacto.

## 📂 Estructura de Archivos

- `index.html`: Contiene el maquetado principal del sitio, los textos y los enlaces de contacto.
- `style.css`: Incluye todos los estilos visuales (colores, fondos, tipografías) y el sistema de animaciones.
- `SVG/`: Directorio que contiene los recursos gráficos de la marca.
  - `logo-amari.svg`: El logotipo principal en color amarillo que se está usando en la interfaz.
  - `iso.svg`: El isotipo utilizado como icono de la pestaña del navegador (Favicon).

## 🎨 Características de la Interfaz

1. **Fondo Inmersivo:** Utiliza un gradiente radial (`radial-gradient`) para simular un efecto de viñeta fotográfica, oscureciendo los bordes y resaltando el logo en el centro.
2. **Animaciones Dinámicas:**
   - **SlideUp & Fade:** Los textos y botones entran deslizando hacia arriba en cascada al cargar la web.
   - **Flotación:** El logotipo principal tiene un movimiento sutil de flotación continua.
   - **Hover effects:** Los botones de contacto tienen efectos estilo *glassmorphism* (cristal esmerilado) que se iluminan al pasar el cursor.
   - **Esquinas decorativas:** Bordes asimétricos que aparecen suavemente simulando el marco del diseño original.
3. **Contactos Directos:**

   - Enlace `mailto:` configurado hacia `hola@morehdmkt.com`.

## 🛠️ Cómo visualizar y editar el sitio

- **Para ver la web:** Simplemente haz doble clic sobre el archivo `index.html`. Se abrirá en tu navegador web por defecto de manera local.
- **Para modificar un texto:** Abre el archivo `index.html` con tu editor de código o bloc de notas, busca el texto (como "Próximamente" o la descripción) y guárdalo.
- **Para cambiar los colores:** Abre `style.css` y modifica las variables CSS que se encuentran al principio del archivo bajo la etiqueta `:root` (ej. `--accent-color`).
